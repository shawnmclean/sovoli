# TypeScript Guide

Essential concepts and patterns for TypeScript development.

## Basic Types

TypeScript provides several basic types for common values.

```typescript
// Boolean
let isDone: boolean = false;

// Number
let decimal: number = 6;
let hex: number = 0xf00d;

// String
let color: string = "blue";

// Array
let list: number[] = [1, 2, 3];
let list2: Array<number> = [1, 2, 3];
```

## Interfaces

Interfaces define the shape of objects.

```typescript
interface User {
  id: number;
  name: string;
  email?: string; // Optional property
}

function createUser(user: User): User {
  return user;
}
```

## Types vs Interfaces

### Interfaces

- Can be extended
- Can be implemented by classes
- Support declaration merging

```typescript
interface Animal {
  name: string;
}

interface Dog extends Animal {
  breed: string;
}
```

### Types

- Can represent unions, primitives, and computed types
- Cannot be extended
- More flexible for complex types

```typescript
type Status = "loading" | "success" | "error";

type UserWithStatus = User & {
  status: Status;
};
```

## Generics

Generics allow you to create reusable components.

```typescript
function identity<T>(arg: T): T {
  return arg;
}

let output = identity<string>("myString");
let output2 = identity<number>(123);
```
